# 快速开始

引导您如何在项目中使用 Tu View Plus。

## 1.安装组件库

```bash
npm i tu-view-plus
```

## 2.引用组件库

> 在 main.js 中引入组件库

```javascript
// 全部引入
import TuView from 'tu-view-plus';
Vue.use(TuView);

// 按需引入
import { TuButton } from 'tu-view-plus';
Vue.use(TuButton);
```
