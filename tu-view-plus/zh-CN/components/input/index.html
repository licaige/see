Input 输入框 ​

基本表单组件，并在原生控件基础上进行了功能扩展，可以组合使用。

基础用法 ​

输入框的基本用法。

<template>
  <tu-input v-model="inputValue" placeholder="请输入" />
</template>

<script lang="ts" setup>
import { ref } from 'vue';
const inputValue = ref('');
</script>

禁用状态 ​

通过 disabled 属性指定是否禁用 input 组件。

<template>
  <tu-input v-model="inputValue" disabled placeholder="请输入" />
</template>

<script lang="ts" setup>
import { ref } from 'vue';
const inputValue = ref('');
</script>

可清空 ​

使用 clearable 属性即可得到一个可清空的输入框。

<template>
  <tu-input v-model="inputValue" placeholder="请输入" clearable />
</template>

<script lang="ts" setup>
import { ref } from 'vue';
const inputValue = ref('');
</script>

格式化 ​

使用 formatter 属性显示格式化内容，通常需要搭配 parser 属性一起使用。

<template>
  <tu-input
    v-model="inputValue"
    placeholder="请输入"
    :formatter="(value: string | number) => `$ ${value}`.replace(/\B(?=(\d{3})+(?!\d))/g, ',')"
    :parser="(value: string) => value.replace(/\$\s?|(,*)/g, '')"
  />
</template>

<script lang="ts" setup>
import { ref } from 'vue';
const inputValue = ref('');
</script>

密码框 ​

使用 show-password 属性即可得到一个可切换显示隐藏的密码框。

<template>
  <tu-input
    v-model="inputValue"
    type="password"
    placeholder="请输入"
    show-password
  />
</template>

<script lang="ts" setup>
import { ref } from 'vue';
const inputValue = ref('');
</script>

带 icon 的输入框 ​

带有图标标记输入类型。

属性方式

slot方式

<template>
  <p>属性方式</p>
  <tu-row :gutter="20" class="mb-2">
    <tu-col :span="12">
      <tu-input
        v-model="inputValue1"
        placeholder="请输入"
        :suffix-icon="Search"
      ></tu-input>
    </tu-col>
    <tu-col :span="12">
      <tu-input
        v-model="inputValue2"
        placeholder="请输入"
        :prefix-icon="Search"
      ></tu-input>
    </tu-col>
  </tu-row>
  <p>slot方式</p>
  <tu-row :gutter="20">
    <tu-col :span="12">
      <tu-input v-model="inputValue3" placeholder="请输入">
        <template #suffix>
          <tu-icon><Search /></tu-icon>
        </template>
      </tu-input>
    </tu-col>
    <tu-col :span="12">
      <tu-input v-model="inputValue4" placeholder="请输入">
        <template #prefix>
          <tu-icon><Search /></tu-icon>
        </template>
      </tu-input>
    </tu-col>
  </tu-row>
</template>

输入长度限制 ​

maxlength 和 minlength 属性，用来限制输入框的字符长度。

0 / 10

字符长度是用 Javascript 的字符串长度统计的。对于类型为 text 的输入框，在使用 maxlength 属性限制最大输入长度的同时，可通过设置 show-word-limit 属性来展示字数统计。

<template>
  <tu-input
    maxlength="10"
    type="text"
    v-model="inputValue"
    placeholder="请输入"
    show-word-limit
  />
</template>

<script lang="ts" setup>
import { ref } from 'vue';
const inputValue = ref('');
</script>

不同尺寸 ​

tu-input 组件提供除了默认值 medium 以外的三种尺寸。

额外的尺寸：large、small、mini，通过设置 size 属性来配置它们。

<template>
  <tu-row :gutter="20" class="demo-input-row">
    <tu-col :span="6">
      <tu-input v-model="inputValue1" size="mini" placeholder="请输入" />
    </tu-col>

    <tu-col :span="6">
      <tu-input v-model="inputValue2" size="small" placeholder="请输入" />
    </tu-col>

    <tu-col :span="6">
      <tu-input v-model="inputValue3" placeholder="请输入" />
    </tu-col>

    <tu-col :span="6">
      <tu-input v-model="inputValue4" size="large" placeholder="请输入" />
    </tu-col>
  </tu-row>
</template>

<script lang="ts" setup>
import { ref } from 'vue';

const inputValue1 = ref('');
const inputValue2 = ref('');
const inputValue3 = ref('');
const inputValue4 = ref('');
</script>

Input Events ​

事件名描述参数
blur在 Input 失去焦点时触发Function
focus在 Input 获得焦点时触发Function
change仅在输入框失去焦点或用户按下回车时触发Function
input在 Input 值改变时触发Function
clear在点击由 clearable 属性生成的清空按钮时触发Function

Input Slots ​

参数名描述
prefix输入框头部内容，只对 type 为 text 有效
suffix输入框尾部内容，只对 type 为 text 有效
prepend输入框前置内容，只对 type 为 text 有效
append输入框后置内容，只对 type 为 text 有效
