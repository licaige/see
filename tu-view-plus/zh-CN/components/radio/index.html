Radio 单选框 ​

何时使用 ​

用户要从一个数据集中选择单个选项，且能并排查看所有可选项，选项数量在 2~7 之间时，建议使用单选按钮。

基础用法 ​

由于选项默认可见，不宜过多，若选项过多，建议使用 Select 选择器。

要使用 Radio 组件，只需要设置 v-model 绑定变量，选中意味着变量的值为相应 label 属性的值，label 可以是 String、Number 或 Boolean。

<template>
  <tu-radio v-model="radio" :label="1">选项1</tu-radio>
  <tu-radio v-model="radio" :label="2">选项2</tu-radio>
</template>

<script lang="ts" setup>
import { ref } from 'vue';

const radio = ref(1);
</script>

禁用状态 ​

单选框不可用的状态。

只要在 tu-radio 元素中设置 disabled 属性即可，它接受一个 Boolean，true 为禁用。

<template>
  <tu-radio v-model="radio" :label="1" disabled>选项1</tu-radio>
  <tu-radio v-model="radio" :label="2" disabled>选项2</tu-radio>
</template>

<script lang="ts" setup>
import { ref } from 'vue';

const radio = ref(1);
</script>

单选框组 ​

适用于在多个互斥的选项中选择的场景。

结合 tu-radio-group 元素和子元素 tu-radio 可以实现单选组，在 tu-radio-group 中绑定v-model，在 tu-radio 中设置好 label 即可，无需再给每一个 tu-radio 绑定变量，另外，还提供了 change 事件来响应变化，它会传入一个参数 value。

<template>
  <tu-radio-group v-model="radio" @change="handleChange">
    <tu-radio :label="1">选项1</tu-radio>
    <tu-radio :label="2">选项2</tu-radio>
    <tu-radio :label="3">选项3</tu-radio>
  </tu-radio-group>
</template>

<script lang="ts" setup>
import { ref } from 'vue';

const radio = ref(1);

function handleChange(value: number) {
  console.log(value);
}
</script>
