<h3 align="center" style="margin: 30px 0 30px;font-weight: bold;font-size:40px;">TuViewPlus</h3>
<h3 align="center">一款有触觉的 Vue3.0 UI组件库</h3>

### 快速开始

#### 1.安装组件库

```bash
npm i tu-view-plus
```

#### 2.引用组件库

> 在 main.js 中引入组件库

```javascript
// 全部引入
import TuViewPlus from 'tu-view-plus';
Vue.use(TuViewPlus);

// 按需引入
import { TuButton } from 'tu-view-plus';
Vue.use(TuButton);
```
