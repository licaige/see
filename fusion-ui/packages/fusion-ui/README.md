## 特性

- 🧜 Fusion Ui 提供了整洁和美丽的 UI 组件
- 🎡  按需引入-提供解析器以自动导入仅使用过的组件。
- 💪 支持 TypeScript-支持类型脚本和类型检查及类型推断。
- ⚡️ 全面基于 Vite，速度够快
- 📃 详细的中/英文档
- 🛠  More...
