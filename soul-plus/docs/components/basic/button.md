## Button 按钮

<ClientOnly>
  <doc-button/>
</ClientOnly>

### 组件属性

| 名称       | 类型      | 描述                                                |
| ---------- | --------- | --------------------------------------------------- |
| `loading`  | `Boolean` | 加载状态                                            |
| `disabled` | `Boolean` | 禁用状态                                            |
| `type`     | `String`  | 类型，可选值 `primary` `success` `warning` `danger` |
| `plain`    | `Boolean` | 是否朴素按钮                                        |
| `round`    | `Boolean` | 是否圆角                                            |
| `block`    | `Boolean` | 是否块级按钮                                        |
