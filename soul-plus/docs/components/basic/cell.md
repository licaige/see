## Cell 单元格

<ClientOnly>
  <doc-cell/>
</ClientOnly>

### 组件属性

| 名称        | 类型        | 描述             |
| ----------- | ----------- | ---------------- |
| `title`     | `String`    | 标题             |
| `value`     | `String`    | 单元格右侧内容   |
| `label`     | `String`    | 次要标题         |
| `clickable` | `clickable` | 单元格是否可点击 |
